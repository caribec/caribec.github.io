Volunteer Management System

A full-stack web app (React, Node/Express, MySQL) that helps organizations create events and track volunteer participation. Volunteers can browse events, join/leave, and see a detailed Volunteer History with event info and status. The backend enforces field/type/length validation with REST APIs and unit tests, and the schema models users, profiles, events, and participation records.
